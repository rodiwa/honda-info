# Honda Info App
Generic info about a certain Honda.

## Dev Corner
### Running locally
- `docker-compose up -d`
- Open `localhost:3000` in browser
- `docker-compose down`

### Runnin on prod
- `docker-compose -f docker-compose-prod.yml up -d`
- Open `localhost` in browser
- `docker-compose -f docker-compose-prod.yml down`
